# Working example of a vue TS app working in a monorepo with yarn v2

## Instructions
1. Clone the repo
2. Run `yarn`
3. If you use VSCode `yarn pnpify --sdk`
4. Run `yarn workspace example-panel serve`
5. Watch this beauty come alive!
